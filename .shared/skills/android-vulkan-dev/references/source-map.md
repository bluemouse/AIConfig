# Source map

Use these sources when updating or checking this skill. Prefer official Android documentation for facts that may change.

## Official Android Vulkan/NDK documentation

- Vulkan graphics API on Android: https://developer.android.com/ndk/guides/graphics
- Get started with Vulkan: https://developer.android.com/ndk/guides/graphics/getting-started
- Vulkan design guidelines: https://developer.android.com/ndk/guides/graphics/design-notes
- Vulkan shader compilers: https://developer.android.com/ndk/guides/graphics/shader-compilers
- Vulkan validation layers: https://developer.android.com/ndk/guides/graphics/validation-layer
- Vulkan extensions on Android: https://developer.android.com/ndk/guides/graphics/extensions
- Android Vulkan Profiles: https://developer.android.com/ndk/guides/graphics/android-vulkan-profile
- Wide color gamut: https://developer.android.com/training/wide-color-gamut
- NDK build/CMake/JNI basics: see the `android-ndk-dev` source map.

## Cross-skill boundaries

- `android-dev`: app-layer Kotlin, Compose, SDK camera/media, permissions, architecture, release quality.
- `android-ndk-dev`: C++ integration, CMake, JNI, native memory/threading, native libraries, native debug/profiling.
- `android-vulkan-dev`: Vulkan renderer internals, shaders, synchronization, surfaces, GPU image processing, validation, and GPU performance.
