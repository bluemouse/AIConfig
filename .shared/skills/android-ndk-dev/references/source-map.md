## Cross-skill boundaries

- `android-dev`: app-layer Kotlin, Compose, SDK camera/media, permissions, architecture, release quality.
- `android-ndk-dev`: C++ integration, CMake, JNI, native memory/threading, native libraries, native debug/profiling.
- `android-vulkan-dev`: Vulkan rendering and GPU image processing through the NDK.
